Technical Manual

The information in this document is provided with the purpose of illustrating the structure of the IOT Project, which includes the process covered to create it, the task model of the project which illustrate the abstract functionality of the system, the activities and process of each activity.

The document includes UML diagrams with software pattern applied, in addition it includes the database modelling and a quick view to API system on flask.
